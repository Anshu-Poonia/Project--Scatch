This is a backend project.
In which we uses the ejs for the UI-interface, mongoDB for the database, express.js for the router creation and node.js for the server connection.

Include = Node.js , Express.js and MongoDB

install to run the code : npm i express ejs mongoose jsonwebtoken bcrypt path cookie-parser

About the project :
     - It's a ecommerce website
     - Selling the bags of different type
     - Including the admin panel and user panel with authentication and authorization
     - In this project admin create the products and set the price, discount and desogn of the product panel
     - And this project display on the user panel
     - From user panel user select the product and add to the cart where price and all other stuff display
