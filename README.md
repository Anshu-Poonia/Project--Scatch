|  About the project :                                                                                                                                             |
|       - It's a ecommerce website.                                                                                                                                |
|       - Selling the bags of different type.                                                                                                                      |
|       - Including the admin panel and user panel with authentication and authorization.                                                                          |
|       - In this project admin create the products and set the price, discount and design of the product panel.                                                   |
|       - And this product display on the user panel.                                                                                                              |
|       - From user panel user select the product and add to the cart where price and all other stuff display.                                                     |
|       - Some more functionalities also added in which you can sort the products by using the filters like discount, newest, cheapest, expensive and others.      |
